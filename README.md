# Pharmatrack-mobile

Sistema de Gestión Logística Farmacéutica desarrollado para optimizar el control de pedidos, inventario, lotes, evidencias y distribución de productos farmacéuticos.

## Objetivo

Digitalizar los procesos logísticos de Pharma-C mediante una plataforma compuesta por:

- Aplicación Android para almacén.
- Panel Web para logística y contabilidad.
- API REST en PHP.
- Base de datos MySQL.

## Tecnologías

### Backend
- PHP 8
- MySQL
- API REST

### Web

- Bootstrap 5
- JavaScript

### Mobile
- Android Studio
- Java
